# GLEngine
Simple Game Renderer for Learning

Systerm:

*Win10 x86
*Visual Studio 2015

Library:

* GLEW
* SDL
* SDL Image 
* ASSIMP
* Freetype

## Features

* Mesh loading (completed)
* FPS Camera / Player Camera (completed)
* Height Terrain (completed)
* Detect Terrain (completed)
